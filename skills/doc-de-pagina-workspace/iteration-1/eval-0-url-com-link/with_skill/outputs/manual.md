# Manual do Usuário — Cadastro de Segurado

**Módulo:** Cadastro | **Versão:** 1.0 | **Data:** 26/03/2026
**Elaborado por:** VSeguradora — LPL Solutions

---

## 1. Visão Geral

O módulo de **Cadastro de Segurado** permite que você registre novas pessoas que desejam contratar serviços de seguro com a VSeguradora.

**A quem se destina:**
- Gerentes de vendas
- Assistentes de cadastro
- Operadores de atendimento

**O que você consegue fazer:**
- Registrar dados pessoais do segurado
- Validar informações obrigatórias
- Salvar o cadastro no sistema

**Pré-requisitos:**
- Acesso ao menu "Cadastro" no sistema
- Documento de identidade (CPF) válido do segurado
- Dados pessoais corretos do candidato a segurado

---

## 2. Passo a Passo de Uso

### 2.1 Acessando a Tela de Cadastro

1. No menu principal, clique em **"Cadastro"**
2. Selecione a opção **"Novo Segurado"**
3. A tela de cadastro será aberta

### 2.2 Preenchendo os Campos

Preencha os campos conforme descrito na tabela abaixo:

| Campo | Tipo | Obrigatório | Descrição |
|-------|------|-------------|-----------|
| **Nome do Segurado** | Texto | Sim | Nome completo da pessoa que será segurada |
| **CPF** | Texto | Sim | Número do CPF sem pontos ou barras (11 dígitos) |
| **Data de Nascimento** | Data e Hora | Sim | Data de nascimento no formato DD/MM/AAAA |
| **Telefone** | Texto | Não | Telefone para contato (com ou sem parênteses) |
| **E-mail** | Texto | Não | Endereço de e-mail válido para comunicações |

### 2.3 Validações Automáticas

O sistema verifica automaticamente as seguintes condições:

- **CPF válido**: O CPF deve estar correto e ser um número válido
- **Formato de e-mail**: Se fornecido, o e-mail deve conter um @ e um domínio válido
- **Idade mínima**: O segurado deve ter no mínimo 18 anos de idade

Se algum campo estiver incorreto, o sistema exibirá uma mensagem de erro em vermelho indicando o problema.

### 2.4 Salvando o Cadastro

1. Após preencher todos os campos obrigatórios, clique no botão **"Salvar"**
2. O sistema validará os dados
3. Se tudo estiver correto, a mensagem **"Cadastro realizado com sucesso"** aparecerá
4. O segurado será adicionado ao sistema e poderá contratar serviços de seguro

### 2.5 O que Fazer em Caso de Erro

Se aparecer uma mensagem de erro:

1. Leia a mensagem com atenção — ela indicará qual campo está com problema
2. Corrija o dado no campo indicado
3. Clique novamente em **"Salvar"**

## 3. Perguntas Frequentes (FAQ)

**P: Posso cadastrar alguém com menos de 18 anos?**
R: Não. O sistema não permite cadastrar segurados menores de 18 anos. Você deve informar a data de nascimento correta.

**P: O CPF é obrigatório?**
R: Sim. O CPF é um campo obrigatório e deve ser um CPF válido. O sistema verifica automaticamente se o número é válido.

**P: Preciso incluir pontos e barras no CPF?**
R: Não. Digite apenas os 11 dígitos do CPF, sem pontuação. O sistema aceitará com ou sem formatação.

**P: E se a pessoa não quiser fornecer telefone ou e-mail?**
R: Telefone e e-mail são opcionais. Você pode deixar em branco e salvar apenas com os dados obrigatórios (nome, CPF e data de nascimento).

**P: Posso editar um cadastro depois de salvo?**
R: Este manual cobre apenas o cadastro inicial. Para editar um cadastro existente, consulte a documentação sobre "Edição de Cadastro".

**P: O que significa a mensagem "CPF inválido"?**
R: Significa que o CPF digitado não passou na validação do sistema. Verifique se digitou corretamente os 11 dígitos.

**P: Qual é o formato correto de um e-mail?**
R: Um e-mail válido tem a estrutura: `nome@dominio.com`. Exemplo: `joao@example.com`. Deve ter um @ e um domínio com extensão.

**P: E se eu cometer um erro ao digitar? Posso corrigir?**
R: Sim! Você pode alterar qualquer campo antes de clicar em "Salvar". Se já tiver salvado, a edição deve ser feita pela tela de edição de cadastros.

**P: Quanto tempo leva para o cadastro ser processado?**
R: O cadastro é salvo instantaneamente quando você clica em "Salvar" e recebe a mensagem de sucesso.

**P: Preciso de permissões especiais para cadastrar segurados?**
R: Você precisa ter acesso ao menu "Cadastro" no sistema. Se não conseguir acessá-lo, entre em contato com seu supervisor.
